## Remaining Issues

### HIGH Priority

**H1. Privacy page references barkingdolphin.com** (new)
`privacy.html` line 153: "Where necessary, barkingdolphin.com uses cookies..." — template leftover. The site is tynehamvillage.org. Credibility risk if anyone reads it.

**H2. Church door note attribution inconsistency** (old H8)
FAQ answer 1 says "W.H. Bond on behalf of the families", FAQ answer 2 says "on behalf of the women and children", Quotation schema says "families", blockquote footer says "The Women and Children of Tyneham". Choose one consistent attribution.

**H3. llms.txt Bond family tenure wrong**
Line 25: "The squires of Tyneham for **over 400 years**" — should be ~260 years. Causes AI contradiction with the bond-family page.

**H4. About page still thin for E-E-A-T**
~400 words, no author photo, no external validation links. Target 500+ words with photo and credentials.

**H5. Person schema missing sameAs**
About page Person schema has no `sameAs` property — critical for AI entity trust. Add LinkedIn, Wikipedia, or Wikidata link.

### MEDIUM Priority

**M1. Content-Security-Policy header missing**
Absent from `_headers`. Bootstrap CDN + Google AdSense makes a permissive CSP feasible.

**M2. CSS unminified**
240KB unminified — Cloudflare Auto Minify would fix in 2 minutes.

**M3. Contact page missing ContactPoint schema**
Only has BreadcrumbList. Should add ContactPoint or Organization schema.

**M4. Tyneham church title uses hyphen instead of pipe**
`<title>Tyneham Church - St Mary's...</title>` — all other pages use pipe: `Page Title | Tyneham Village`.

**M5. Oversized images**
38 files >500KB. `wildflowers.png` is 3.4MB — should be WebP.

**M6. Google Fonts render-blocking**
Loaded via CSS `@import` instead of `<link>` with `preconnect` + `font-display: swap`.

**M7. Direct WebP src on ~20 images**
Some `<img src="image.webp">` without `<picture>` fallback — older browsers unsupported.

### LOW Priority

**L1. Expand thin pages** — lulworth-cove (~750 words), tyneham-church (~700 words) competing with National Trust and English Heritage.

**L2. Publish draft pages** — tyneham-in-wartime, corfe-castle-walk, kimmeridge-tyneham-walk are complete and ready.

**L3. Add srcset for responsive images** — especially the 3072px monsters.

**L4. Fix worbarrow-bay-today.webp filename** — double 'r' typo.

**L5. Bond tenure: 250 vs 260 years** — bond-family says "nearly 250 years", corfe-castle says "nearly 260 years". Minor, but harmonise.
